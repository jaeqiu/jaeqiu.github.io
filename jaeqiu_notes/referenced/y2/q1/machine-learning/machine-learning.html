---------------------------Clustering-------Cut

Selecting k is difficult.

Agglomerative:
  • Bottom-up
  • Each point as cluster
  • Merge closest clusters
  • Repeat until condition (like #clusters<1)

What is "closest"?

  • point vs point: just compare distance (euclidean, manhattan, etc)
  • cluster vs * : represent the cluster with the
    • furthest in-cluster object (linkage=single)
    • closest in-cluster object (linkage=complete)
    • average in-cluster object (linkage=mean)

Divisive:
  • Top-down
  • all points in one cluster
  • split clusters
  • repeat until condition

Implementation: apply k means, for each of k clusters, apply k means (keep splitting)

PCA

Map data onto linear subspace, maximising projected dataset variance

Define Principal components (new vectors):

  • PC1: Direction of greatest variability
  • PC2: Out of perpendicular directions, direction of greatest variability
    The first m components become m new dimensions

Change coordinates of each data point to new dimensions
